Security & Control

ConvOps is an AI agent with cloud access. Here's how we keep it safe:

✅ Read-only by default — you explicitly grant write permissions per resource
✅ Approval workflow — ALL destructive actions require human confirmation
✅ Full audit log — every command, approval, and execution logged
✅ AWS IAM roles — no long-lived credentials. Session tokens only.
✅ Encrypted everything — TLS in transit, AES-256 at rest
✅ Your AWS account — IAM role + SNS topic deploy into your account. No credentials ever leave your cloud.
